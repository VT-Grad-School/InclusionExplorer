Technical Information

This application heavily utilizes d3, having referenced the Force-Directed Graph example, to build the interactive visualization. We employ AngularJS to manage the front-end routing and to bind the data to the templates. We (will) use Node.js with the Jade Template Engine (for server-side rendering).
